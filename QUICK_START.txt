🔧 USE IT IN YOUR APP
═══════════════════════════════════════════════════════════════════

1. Add to your code:

   import "github.com/yourusername/ltt/pkg/exporter"

   func main() {
       exp, _ := exporter.New(exporter.DefaultConfig())
       defer exp.Shutdown(context.Background())

       tp := sdktrace.NewTracerProvider(
           sdktrace.WithBatcher(exp),
       )
       otel.SetTracerProvider(tp)

       // Your traced code here...
   }

2. Run your app

3. In another terminal: ./ltt

🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

Issue: "Connection refused"
Fix: Start exporter first (test_e2e.go or your app)

Issue: "No spans showing in TUI"
Fix: Wait 1-2 seconds for buffering, or press 'r' to refresh

Issue: "Can't build"
Fix: go mod download && go mod tidy

Issue: "Tests failing"
Fix: Make sure socket port 9090 is free
